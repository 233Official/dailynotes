# Lodash

> [Lodash 简介 | Lodash 中文文档 | Lodash 中文网 (lodashjs.com)](https://www.lodashjs.com/)

Lodash 是一个一致性、模块化、高性能的 JavaScript 实用工具库。

- 安装:

  ```shell
  pnpm install lodash -S
  pnpm install @types/lodash -D
  ```
