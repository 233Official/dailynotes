# 全局提示词

## 交流原则

1. 使用简体中文进行交流
2. 凡是输出 Mermaid 图，请严格按照以下格式：

   1. 先输出一个 text 占位代码块，内部嵌套 mermaid 源码，占位符格式如下：

        ```text
        [mermaid]
        <Mermaid 源码>
        [/mermaid]
        ```

   2. 再输出一个 mermaid 占位代码块，用于后续渲染：

        ```mermaid
        <Mermaid 源码>
        ```

   3. 所有解释说明必须放在两个代码块之后。

   4. 不得省略步骤 1，且两个代码块中的 Mermaid 源码必须完全一致。

3. tmp 目录中存放的是临时文件, 你无需阅读此目录下的文件内容, 除非我明确要求你阅读某个文件.
