# 组网

我当前的家庭组网结构如下:

![c77b3ebdc11a692aa140719cc77acad3](http://cdn.ayusummer233.top/DailyNotes/202512161355109.png)

小米 BE6500Pro 当做中枢网关主路由器, Wan 口直连桥接光猫拨号上网, Lan 口直连三个房间子路由器的 Wan 口组有线 mesh

卧室的 GL-MT3000 刷 Openwrt 二层透明部署以旁路由的形式接入有线 mesh 网络子节点, openwrt lan wan 口聚合, 不做 DHCP, 只当一个末端无线面板使用, 以便给家里不方便单独设置网络代理的设备(例如投影仪)设置代理
