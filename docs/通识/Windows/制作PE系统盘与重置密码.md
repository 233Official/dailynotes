# Windows 制作PE系统盘与重置密码

**文档说明**：本指南主要用于在忘记 Windows 开机密码时，通过制作微PE（WePE）启动盘进入预安装环境，以清除密码或绕过权限的方式，安全地救援电脑内的重要数据。

> ⚠️ **重要提示**：此方法仅适用于未开启 BitLocker 磁盘加密的电脑。如果 C 盘带有金色小锁图标，强制清除密码将导致数据永久锁定。

---

## 🛠️ 准备工作

在开始之前，你需要准备以下工具：

- **一台可正常使用的 Windows 电脑**（用于制作启动盘）。
- **一个 U 盘**：容量建议 8GB 或以上（**注意：制作过程会清空 U 盘，请提前备份重要数据！**）。
- **微PE工具箱（WePE）**：前往官网 (wepe.com.cn) 下载 V2.3 或 V2.4 版本的 64位 (x64) 安装程序。

------

## 🚀 第一步：制作 PE 启动盘

1. 将准备好的 U 盘插入正常使用的 Windows 电脑并**备份 U 盘中还需要保存的文件**

2. 双击运行下载好的 `WePE64_V2.3.exe`。

3. **关键选择**：在软件界面右下角，点击第一个图标（带有 U 盘形状），选择**“安装 PE 到 U 盘”**。

   > 这一步会格式化 U 盘，请提前备份好数据

4. **配置参数**（强烈建议保持默认的最优解）：

   - **待写入 U 盘**：确保选中了你的目标 U 盘。

   - **安装方法**：方案一（UEFI/Legacy 全能三分区）—— 兼容新老电脑，两个启动分区是**隐藏**的可以防误删。

     这个方案会把 U 盘分为三个部分：

     - **EFI 引导分区**（隐藏）：专门用于启动 **Windows 11/10** 新电脑 (UEFI 模式)。
     - **Legacy 引导分区**（隐藏）：专门用于启动**老旧电脑** (BIOS 模式)。
     - **数据分区**（可见）：剩下的空间可以当普通 U 盘用，存文件。

     除非这台 U 盘在某些奇葩的主板上无法启动，否则**完全不需要**考虑其他选项的方案。它们大多是为了兼容极老旧的硬件或特殊需求设计的。

   - **格式化**：选择 **exFAT** —— 支持大于 4GB 的单文件（如系统镜像），且对 Mac 和 Windows 均友好。相比 NTFS，它对 U 盘（闪存）的损耗也更小。

     其他选项：

     - **NTFS** 是 Windows 的标准硬盘格式。稳定，但对 Mac 不友好（需要第三方软件才能写入），且频繁读写日志对 U 盘寿命有轻微影响。
     - **FAT32**： 兼容性最好，但**存不了大于 4GB 的文件**。在这个时代已经不太推荐作为主力数据盘格式了。

   - **引导模式**：USB-HDD（兼容性最好的主流模式）。把 U 盘模拟成一块**硬盘**。这是过去 15 年来几乎所有电脑的标准启动方式。

     其他选项：

     - **USB-ZIP**： 把 U 盘模拟成一个**大软盘 (ZIP Drive)**。这是 2005 年以前的上古电脑（比如奔腾 4 时代）才需要的技术，现在已经淘汰。

   - **其他勾选选项**

     - **包含 DOS 工具箱**：建议勾选。

     包含一些在纯 DOS 下运行的工具（如内存检测、硬盘低格）。虽然 Win11 用不到，但万一电脑坏得很彻底，这些工具有奇效。

     - **个性化盘符图标**：纯美化，勾不勾无所谓。
     - **同时复制安装包**：勾选后会把 `WePE...exe` 这个安装文件也拷进 U 盘里，方便以后把这个工具分享给别人。

5. 点击**“立即安装进 U 盘”**，确认清空数据警告，等待进度条走完。

------

## 💻 第二步：引导目标电脑进入 PE 系统

1. 将制作好的 PE 启动盘插入那台忘记密码的 Windows 电脑。
2. 确保电脑处于**完全关机**状态。
3. 按下开机键，**立刻连续敲击启动热键**（通常是 `F12`、`F11` 或 `Esc`，具体取决于电脑品牌）。
4. 在弹出的启动菜单 (Boot Menu) 中，找到带有 **"USB"**、你的 U 盘品牌名或 **"UEFI"** 字样的选项，按回车进入。
5. 等待加载，进入类似 Windows 桌面的微PE系统。
   - *测试：双击打开“此电脑”，尝试访问 C 盘，确认能看到里面的文件且没有 BitLocker 锁。*

------

## 🔓 第三步：清除或绕过系统密码

进入 PE 后，打开桌面上的 **Dism++** 工具（或“Windows 密码修改” NTPWEdit），进入**账户管理**界面。根据账号类型，分为两种情况：

---

### 情况 A：本地账户密码清除

如果目标账户是普通的本地账号：

1. 在用户列表中选中需要解锁的用户名。
2. 点击下方的**“清空密码”**（或修改密码留空）。
3. 点击**“保存更改”**。 *(原理：此操作仅修改了系统门锁 `SAM` 文件，将密码记录抹除，绝对不会触碰任何用户资料和文件。)*

---

### 情况 B：微软在线账号绕过 (推荐方案)

如果目标账户绑定了微软邮箱，PE 环境下无法直接修改云端密码，此时需要“曲线救国”：

> 如果你是这个微软邮箱的所有者的话
>
> - 如果还记得密码可以找找修改登入方式的资料，改成使用微软账户登入然后修改 PIN 码
> - 如果忘记了邮箱密码的话可以登录微软官网重置密码

1. 在用户列表中，找到系统内置的最高权限账号：`Administrator`。
2. 选中 `Administrator`，点击**“启用”**（确保其状态从“禁用”变为“启用”）。
3. 为保险起见，选中 `Administrator` 后再次点击**“清空密码”**。
4. 点击**“保存更改”**，关闭软件。

------

## 📂 第四步：重启并提取文件

1. 拔掉 U 盘，正常重启电脑。
2. **如果是情况 A**：直接点击原用户头像，无需密码即可进入桌面。
3. **如果是情况 B**：
   - 在登录界面左下角，选择新出现的 `Administrator` 账户。
   - 直接点击登录（无密码）进入全新桌面。
   - 打开“此电脑” -> `C:\Users (用户)` -> 找到朋友原账号名字的文件夹（首次打开可能需要几秒钟获取权限）。
   - 进入后即可访问原账号的 `Desktop (桌面)`、`Documents (文档)`、`Pictures (图片)` 等所有重要资料。
