  • зӣёеә”зҺҜеўғе‘Ҫд»ӨиЎҢжү§иЎҢ
    pip install TensorFlow --user --no-warn-script-location
    • --user : pip й»ҳи®Өе®үиЈ… package еҲ° system directory, йҖҡиҝҮ --user еҸҜд»Ҙе°Ҷ package е®үиЈ…еҲ° /home и·Ҝеҫ„дёӢ
    • --no-warn-script-location : еҝҪз•Ҙи„ҡжң¬иӯҰе‘Ҡ
    • дёӘдәәдҪҝз”Ёжё…еҚҺзҡ„жәҗе®үиЈ… TensorFlow ж—¶з»Ҹеёёи¶…ж—¶,еҸҜд»ҘдҪҝз”ЁйҳҝйҮҢзҡ„жәҗ
