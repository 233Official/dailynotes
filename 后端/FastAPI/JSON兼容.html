дҫӢеҰӮ, еҰӮжһңжҲ‘д»¬йңҖиҰҒе°Ҷд»–еӯҳе…Ҙж•°жҚ®еә“, FastAPI жҸҗдҫӣдәҶ jsonable_encoder() еҮҪж•°

дҪҝз”Ё jsonable_encoder

жҲ‘д»¬еҒҮи®ҫеҪ“еүҚжҲ‘д»¬жңүдёҖдёӘеҸӘжҺҘеҸ— JSON е…је®№ж•°жҚ®зҡ„ж•°жҚ®еә“ fake_db.

дҫӢеҰӮ, е®ғдёҚжҺҘеҸ— datetime еҜ№иұЎ, еӣ дёәиҝҷдәӣеҜ№иұЎдёҺ JSON дёҚе…је®№

жүҖд»Ҙ, datetime еҜ№иұЎеҝ…йЎ»иҪ¬еҢ–дёәеҢ…еҗ« ISO ж јејҸж•°жҚ®зҡ„ str

еҗҢж ·, иҝҷдёӘж•°жҚ®еә“дёҚдјҡжҺҘж”¶еҲ° Pydantic model(еёҰжңүеұһжҖ§зҡ„еҜ№иұЎ), еҸӘжҺҘж”¶ dict

жҲ‘д»¬еҸҜд»ҘдҪҝз”Ё jsonable_encoder , е®ғжҺҘж”¶дёҖдёӘеҜ№иұЎ, жҜ”еҰӮ Pydantic model, е№¶иҝ”еӣһдёҖдёӘе…је®№ JSON зҡ„зүҲжң¬

Python 3.6~3.10
from datetime import datetime
from typing import Optional

from fastapi import FastAPI
from fastapi.encoders import jsonable_encoder
from pydantic import BaseModel

fake_db = {}


class Item(BaseModel):
    title: str
    timestamp: datetime
    description: Optional[str] = None


app = FastAPI()


@app.put("/items/{id}")
def update_item(id: str, item: Item):
    json_compatible_item_data = jsonable_encoder(item)
    fake_db[id] = json_compatible_item_data
